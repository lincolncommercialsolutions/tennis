# Tennis Game

A simple browser-based tennis (Pong-style) game that works on both desktop and mobile devices.

## How to Play

1. Open `index.html` in a web browser
2. Click or tap the screen to start the game
3. Control your paddle (on the right side) to hit the ball back

### Controls

- **Desktop**: Use ← → arrow keys or A/D keys to move your paddle (← or A moves up, → or D moves down)
- **Mobile**: Touch and drag on the right side of the screen to control your paddle

### Rules

- First player to reach 10 points wins
- Score a point when the ball passes your opponent's paddle
- The ball speeds up slightly with each paddle hit

## Features

- Responsive design that works on any screen size
- Touch support for mobile devices
- AI-controlled opponent
- Score tracking
